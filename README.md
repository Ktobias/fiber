~~~~~~~~
 Models
~~~~~~~~

Building
--------
Name: Name of building. Ex: Hancock Center
Short name: Abbreviation. Ex: HC
Date added: Date and time entry was made.

LanRoom
-------
Building id: Building that LAN room is located in.
LAN room name: Name of LAN room.
Date added: Date and time entry was made.
